# Five Letters API (Old Version)
API bringing five letter words to your platforms

## Endpoints
Home
> http://localhost:1000/?api_key=API_KEY

Random word
> http://localhost:1000/Random?api_key=API_KEY

All words
> http://localhost:1000/Words?api_key=API_KEY
